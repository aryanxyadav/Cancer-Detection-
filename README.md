# AI Cancer Detection Web Application

A full-stack AI-powered application that predicts whether a breast tumor is
Benign or Malignant using Machine Learning, built with Next.js, FastAPI, and
scikit-learn.

## Status: In Progress

- [x] ML pipeline: data loading, preprocessing, EDA
- [ ] Model training & comparison
- [ ] Model evaluation & explainability
- [ ] Backend API (FastAPI)
- [ ] Frontend (Next.js)
- [ ] Deployment
